# Лекция 7. Телевизионные стандарты и форматы вещания

---

## 1. Стандарты телевидения стандартной чёткости (СДТ)

Основные аналоговые стандарты цветного телевидения:

| Стандарт | Регион | Число строк | Частота полей | Полоса видеосигнала |
|----------|--------|-------------|---------------|---------------------|
| **PAL** | Европа (Зап.), Австралия и др. | 625 | 50 Гц | ~5 МГц |
| **SECAM** | Франция, СССР/Россия и др. | 625 | 50 Гц | ~6 МГц |
| **NTSC** | США, Япония | 525 | 60 Гц | ~4.2 МГц |

- Частота поднесущей цветности ~ **6,5 МГц** (зависит от стандарта)
- Используется **чересстрочная развёртка** (2 поля на кадр)

---

## 2. Телевидение высокой чёткости (ТВЧ / HDTV)

Основные параметры HDTV:

| Параметр | Значение |
|----------|----------|
| Общее число строк | **1125** |
| Активных строк | **1080** |
| Соотношение сторон | **16:9** |
| Элементов по горизонтали | **1920** |
| Общее число элементов разложения | **~2 073 600** (1920 x 1080) |
| Частота кадров | **50 / 60** кадр/с |
| Частота строчной развёртки | **33,75 МГц** |

Режимы развёртки:
- **1080i** — чересстрочная развёртка (interlaced)
- **1080p** — прогрессивная развёртка (progressive)

---

## 3. Стандарты цифрового наземного телевещания

Блок-схема на доске иллюстрирует структуру средств наземного цифрового ТВ-вещания в формате ТВЧ.

- Рекомендация **ITU-R BT.656** — стандарт цифрового интерфейса для передачи компонентного видеосигнала
- Совместимость с существующими стандартами (NTSC / PAL)

---

**Ключевые выводы:**
- Переход от СДТ к ТВЧ увеличивает число элементов изображения примерно в **5 раз** (от ~400 тыс. до ~2 млн)
- Формат **16:9** стал стандартом для ТВЧ вместо классического 4:3
- Прогрессивная развёртка (1080p) обеспечивает лучшее качество по сравнению с чересстрочной (1080i), но требует большей полосы пропускания
